# Student Management System

A **Student Management System** built using **Spring Boot**, **Thymeleaf**, **Spring Data JPA**, and **H2 Database**.

This project allows you to manage students: create, read, update, and delete (CRUD) student records through a web interface.

---

## Features

- List all students
- Add a new student
- Edit existing student
- Delete student
- Responsive UI using Bootstrap
- In-memory database (H2) for testing
- Fully implemented with Spring Boot MVC architecture

---

## Tech Stack

- **Backend:** Spring Boot 2.4.5
- **Frontend:** Thymeleaf, Bootstrap 4
- **Database:** H2 (in-memory)
- **Build Tool:** Maven
- **Java Version:** 11+
